Hello World! // by yuisanae2f

많은 것을 혼자서 할 수 있는 Python이나 JavaScript와 달리 C는 혼자서 할 수 있는 게 많이 없어요.
그래서 제일 간단한 예제인 출력도 이런 언어들에 비해 살짝 복잡하죠.
여기서는 가장 간단한 예제인 Hello World!를 출력하는 방법을 보여 줘요.
코드는 이렇게 생겼어요. 
#include <stdio.h>
int main() {
    std::cout << "Hello World!";
    return 0;
}


이 코드를 실행하면 콘솔에 Hello World!를 출력할 거에요.
코드를 클릭해서 어떤 코드가 있는지 확인할 수 있어요!
