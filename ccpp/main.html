자료형

정수형

        

            
bool

            

                1bit의 크기를 가진 이진 정수를 뜻합니다.
                
true

                
참값을 뜻하며, 1의 값을 가집니다.

    
                
false

                
거짓값을 뜻하며, 0의 값을 가집니다.

            

    
            
char

            
1byte의 크기를 가진 정수를 아스키 형태의 문자로 표현한 것을 뜻합니다.

    
            
int

            
4byte의 크기를 가진 정수를 뜻합니다.

        


        
부동 소수형

        

            
float

            
4byte의 크기를 가진 부동 소수점을 뜻합니다.


            
double

            
8byte의 크기를 가진 부동 소수점을 뜻합니다.
