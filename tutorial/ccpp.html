변수 선언

        

            T varName;

            T의 형태로 된 변수 varName을 새로 만듭니다.
        


        
변수 초기화

        

            

                T varName = T.value;
            


            
T.value

            

                T의 형태를 가진 임의의 자료를 T.value 명시했습니다. 

                예를 들어 정수인 int의 경우에는 그 value를 5라 생각할 것입니다. 



                이 경우 정수 형태인 varName을 5로 초기화시키는 방법은 아래와 같을 것입니다. 


                

                    int varName = 5;
                

            

        

    


    
자료형

    

        
자료의 형태를 미리 정의하는 방식입니다.


        
정수형

        

            
bool

            

                1bit의 크기를 가진 이진 정수를 뜻합니다. 

                다른 정수값을 bool로 바꿀 경우, 0일 때 false가 되며, 나머지 경우 전부 true로 간주됩니다.
                
true

                
참값을 뜻하며, 1의 값을 가집니다.

    
                
false

                
거짓값을 뜻하며, 0의 값을 가집니다.

            

    
            
char

            

                
'A'

                

                    1byte의 크기를 가진 정수를 아스키 형태의 문자로 표현한 것을 뜻합니다. 

                    한 글자를 작은 따옴표로 감싸 아래와 같은 형태로 표현할 수 있습니다. 

                    
char var = 'A'


                    한 글자는 하나의 숫자에 치환되므로 여러 개의 문자를 표현하기 위해서는 배열을 사용할 필요가 있습니다.
                


                
문자는 정수가 됩니다.

                

                    문자:

정수는 문자가 됩니다.

Escape Sequence

                

                    

                        문자열 내에서 특정한 의미를 가진 특수한 문자 조합을 나타내는 '\'와 다른 문자로 이뤄진 문자열이나 컴파일러 내에서는 문자로 받아들입니다.
                    

                    \\
                    

                        '\' 그 자체를 뜻합니다. 

                        문자 '\' 자체는 명령으로 쓰이므로 명령 이외에 '\'를 출력하기 위하여 만들어졌습니다.
                    


                    \"
                    

                        문자 '"'를 뜻합니다. 

                        컴파일러가 '"'를 문자열의 시작이라는 명령으로 읽지 않고 문자(char)로써 받아들이게 하기 위함입니다.
                    


                    \'
                    

                        문자 '''를 뜻합니다. 

                        컴파일러가 '"'를 문자의 시작이라는 명령으로 읽지 않고 문자(char)로써 받아들이게 하기 위함입니다. 
                    


                    \n
                    


                    \t
                    


                    \r
                    


                    \b
                    


                    \f
                    


                    \v
                    


                    \a
                    


                    \0
                    


                    \xhh
                    


                    \ooo

포인터

    

        

            모든 변수는 정의된 순간 각자의 메모리를 할당받습니다. 

            그 메모리의 위치 또한 자료형으로 쓰일 수 있으며, 이를 포인터라 부릅니다.
        

        
포인터 초기화

        

            
T* varPoint = &varName;

            

                특정 자료형(여기에서는 T로 표기) 옆에 *를 붙이면 T형식의 자료를 저장할 수 있는 메모리 주소(포인터)로써의 자료형을 뜻합니다. 


                
            

            
T varName

            

                미리 정의된 T 형식의 varName입니다. 


                변수를 새로 정의할 때에 정의된 변수에 새로 메모리를 할당하는데, 그 메모리 주소를 포인터라 부릅니다.

자주 사용하는 인수

            

                int argc
                
입력된 인수들의 개수이자 argv의 길이입니다.

                    아무런 인수가 들어가 있지 않은 경우에도 프로그램의 이름이 인수 그 자체가 되므로 기본값은 1이 됩니다.
                


                char* argv[]
                

                    입력된 인수들은 공백 문자(스페이스바)로 구분된 문자열의 배열로써 argv에 저장됩니다.

<>

        

            시스템 헤더 파일, 즉 컴파일러와 함께 제공되는 헤더 파일을 포함시킬 때 사용됩니다. 

            이 경우 컴파일러는 시스템 헤더 파일을 저장하는 데 사용하는 기본 검색 경로에서 헤더를 찾아 옵니다.
        


        
""

        

            사용자 정의된 라이브러리, 즉 커스텀 헤더 파일을 포함시킬 때 사용됩니다. 

            절대 경로, 혹은 상대 경로에 있는 정보를 읽어 옵니다.
